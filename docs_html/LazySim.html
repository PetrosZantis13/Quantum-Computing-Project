LazySim module¶

This module simulates a circuit with all of the Lazy Matrix implementations

class LazySim.Simulator(gates, register, custom, measurements)¶

Bases: object

simulate()¶

Simulates the quantum circuit by iterating through all of the gates and applyig them one by one. Slower for smaller circuit, but better for larger ones, especially with dense matrices.

Return self.register

(QuantumRegister) The quantum register representing the system

Return self.measurements

(2xn list of lists) Any measurements taken during the simulation
